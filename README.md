## Objectives

- Explain best practices in DevOps and implement basic CI/CD processes

## Project Description
The project focuses on **implementing a CI/CD (Continuous Integration/Continuous Deployment) pipeline** using three main technologies:
- **Git:** Source code version control
- **Jenkins:** Automation server for CI/CD pipeline
- **Docker:** Containerization platform

## Technologies Used
| Technology | Role |
|-----------|---------|
| Git | Version Control System |
| GitHub | Remote Repository |
| Jenkins | CI/CD Automation Server |
| Docker | Container Platform |
| Linux | Operating System |

## Deployment Guide

### 1. Git Configuration
- Install Git on local machine
- Configure user name and email
- Initialize repository

### 2. GitHub Configuration
- Create repository on GitHub
- Push code to remote repository

### 3. Jenkins Configuration
- Install Jenkins server
- Configure Jenkins admin user
- Install necessary plugins
- Create Jenkins pipeline

### 4. Docker Configuration
- Install Docker Engine
- Create Dockerfile for application
- Build and test Docker image

### 5. Jenkins & Docker Integration
- Configure Jenkins to build Docker image
- Set up automated deployment
- Test complete CI/CD pipeline
